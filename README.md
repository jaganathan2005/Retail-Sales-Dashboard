# 🛒 Retail Sales Performance Dashboard

## 📌 Project Overview
Analyzed 9,994 sales orders from a US Superstore (2014–2017)
using Excel, Python, SQL and Power BI.

## 🛠️ Tools Used
- Excel → Data Exploration
- Python (Pandas) → Data Cleaning
- MySQL → SQL Analysis
- Power BI → Dashboard

## 📊 Dataset
- Source: Kaggle Superstore Dataset
- Rows: 9,994 | Columns: 21
- Period: 2014–2017

## 🔍 Key Insights

### 💰 Business Summary
- Total Sales: ₹22,97,200
- Total Profit: ₹2,86,397
- Profit Margin: 12.47%
- Total Unique Orders: 5,009

### 📈 Year wise Trend
- Sales grew 51% from 2014 to 2017
- 2017 = Best year (₹7.3L sales, ₹93K profit)
- 2015 had sales dip but profit increased!

### 🌍 Region Analysis
- West = Best region (14.94% margin)
- Central = Problem region (7.92% margin only)
- South = Hidden gem (better margin than Central!)

### 📦 Category Analysis
- Technology = Star performer (17.4% margin)
- Furniture = Danger zone (2.49% margin only!)
- Office Supplies = Volume king (3742 orders!)

### 💀 Loss Making Sub-Categories
- Tables = -₹17,725 loss (worst!)
- Bookcases = -₹3,472 loss
- Supplies = -₹1,189 loss

### 🎯 Root Cause Found
- Central region giving 80% discounts!
- Binders + Appliances worst affected
- High discount = guaranteed loss proven!

### 🚢 Ship Mode
- Standard Class = 59% of all orders
- First Class = Best margin (13.93%)

## ✅ Business Recommendations
1. 🔴 Cap all discounts at 20% maximum
2. 🔴 Stop heavy discounts in Central region
3. 🔴 Review Tables + Bookcases pricing
4. 🟢 Push Technology products more
5. 🟢 Focus marketing on Home Office segment
6. 🟢 Replicate West region strategy everywhere
